## 変更1：LP セクション2 を全面書き換え

### 旧
セクション名：**共感ブロック**
主読者の親への共感を煽らずに示すブロック。

```
「勉強しなさい」が言えない夜があります。
泥だらけのユニフォームで帰ってきた我が子に、
「勉強しなさい」が言えない。
応援したい気持ちと、心配する気持ち。
二つのあいだで揺れる保護者の方は、少なくありません。
Nobilva は、その毎日に、仕組みでそっと寄り添うサービスです。
```

### 新
セクション名：**サービスの背景・スタンス**
Nobilva の存在意義を強いステートメントで提示するブロック。

```
"学校の成績が足りない" で
チャンスを逃すわけにいかないから。

──────────

野球をがんばる中高生の、将来の選択肢を広げる。
専属メンターが1人ひとりに合わせて、
毎週の計画から日々の実行まで完全サポート。

近年、推薦・内申の比重が上がり、
"成績が足りない" ことで希望進路を諦めるケースが増えています。

練習で疲れて、机に向かう体力が残らない。
遠征で授業を欠席し、戻った頃には授業が先に進んでいる。
気づけば、勉強の時間も気力も、削られていく。

そんな日々を過ごす中高生のために。

Nobilva（のびるば）は、野球に全力で取り組む中高生を、
勉強を「続けられる仕組み」で支える学習サポートです。
```

### 変更の意図
- LPが訴求するのは親だが、サービスを提供するのは子供。「親への共感」に振りすぎていた
- 1文目「"学校の成績が足りない" でチャンスを逃すわけにいかないから。」は **LP最大級のフォントサイズ** で扱う
- 元コピー（旧サイトの強いステートメント）を継承しつつ、ピボット後の文脈（野球特化・時間制約・疲労）に書き換え

### 関連する整合性更新
- LP 0-6 主語ルール運用区分表：セクション2 を「親主語OK」→「中立（背景説明）」に変更
- LP 温度設計図：「2. 共感 [感情導入]」→「2. 背景・スタンス [認知・理解]」に変更
- ビジュアル方針：「窓辺の手元・洗濯物のユニフォーム」→「机に並んだ野球グローブと教科書／練習帰りの選手が机に向かう構図／夕方の自然光が差し込む教室の後ろ姿」（顔なし）

### 影響ファイル
- `nobilva_lp_redesign_v1.md` のみ

## 変更2：紹介手数料制の全面削除

### 旧
チーム経由の紹介に対して、**入会月の月額の5%（仮設定）** をチーム指定口座にお振込みする「紹介手数料制」を提供する設計。

### 新
**紹介手数料制は提供しない。**
チーム経由の紹介に対する経済的還元は、**保護者の月額10%割引（チーム特別価格・仮設定）として直接ご家庭に反映** する形に一本化。

### 変更の意図
- 「保護者の払う料金が割引されるだけ」というシンプルな構造に統一
- お金の流れがチーム側を経由しないため、運用がシンプル・透明

## Claude Code 実装時の注意点

**紹介手数料は完全に提供しない方針です。**

`nobilva_subpage_for_teams_v1.md` や `nobilva_subpage_faq_v1.md` の中で「紹介手数料」という言葉が残っている箇所がありますが、それらはすべて **「提供しない」を明示する否定文脈** のみです。実装時に肯定的な表現に転換しないようにご注意ください。

LP セクション2 のコピーは、1文目「"学校の成績が足りない" で チャンスを逃すわけにいかないから。」が **LP内で最も大きく強い文字サイズ** で扱われる前提のコピー設計です。他のセクションの見出しよりも目立たせてください。
