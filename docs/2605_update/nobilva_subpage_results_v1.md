## 0. 設計の前提

### 0-1. 主読者と温度

LP → このページ／ブログ → このページ／SEO直接の流入も想定。「Nobilvaって本当に成果出るの？」と検証フェーズに入っている人。比較検討の最終段階。

温度は「**疑念→信頼への転換**」。煽らず、誇張せず、淡々と数字と事実を見せる。

### 0-2. このページで達成したいこと

1. **「ヘッドコーチ・代表メンターの個別指導実績」フレーミング** を最初に開示し、Nobilva 直接の実績ではないことを正直に伝える
2. **大きな数字グラフ** で印象的に成果を見せる（誇張なし）
3. **ケーススタディ詳細** で「どんな選手が、どう変わったか」を具体的に伝える
4. **Nobilva 直接生徒の実績** が今後どう積み上がっていくか、誠実に開示
5. 最終的に `/diagnosis` に到達させる

### 0-3. やらないこと

- 確認できない数字・伝聞ベースの実績を載せる
- 顔写真・動画の掲載（外部事情・本人/家族のプライバシー配慮）
- 「他社より◯◯倍」のような比較煽り
- 過剰な装飾（数字を派手にしすぎない）
- 「準備中」「Coming Soon」表記での空きスペース

### 0-4. 言葉づかい

- 必ず明示：**「ヘッドコーチ・代表メンターの個別指導実績」**（Nobilva 直接ではない）
- 数字には必ず併記：**指導期間** と **指導内容**
- 「成績アップ」より「変化」「向上」を選ぶ（誇張感を抑える）
- 仮名表記：「ユウキくん」「アヤカさん」など（実名は使わない）

### 0-5. 主語ルール

LP 0-6 と同じ。ケーススタディ部分は **生徒（仮名）を主語に**。

### 0-6. ビジュアル方針

- 数字はグラフィカルに大きく表示（SVGベース）
- 顔写真なし。代わりに **シルエットアバター** または **小道具イメージ**（バスケットボール・教科書など、Adobe Firefly Web 生成）
- グラフは控えめなブランドカラーで統一（黄 `#f6ce4a` 主色、補助オレンジ `#ee741a`）

### 0-7. 実装条件

- モバイルファースト
- 数字グラフはモバイルでも視認性を保つ（縦並び対応）
- ケーススタディは折りたたみ式（モバイルで長くなりすぎないように）

# セクション3：ケーススタディ詳細 - ユウキくん

ID: `#yuki`

## 目的
ユウキくんの13ヶ月で何が起きたかを物語として伝える。

## レイアウト方針

- 縦長レイアウト
- 写真の代わりにバスケットボールやノートの抽象イメージ（Adobe Firefly Web）
- 時系列のサブ見出し（開始時／3ヶ月後／半年後／1年後）
- 数字グラフを途中に挟む

## コピー

### 見出し
> ユウキくんのケース：中学3年生・バスケットボール部

### 基本情報

| 項目 | 内容 |
|---|---|
| 学年 | 中学3年生（指導開始時：中学2年生） |
| 部活動 | バスケットボール部（週6回練習、土日試合） |
| 指導開始 | 2024年9月 |
| 指導期間 | 13ヶ月 |
| 指導内容 | 週1回のオンライン面談（学習計画）／毎日チャットで進捗管理 |
| 担当 | ヘッドコーチ・中村龍人（個別指導として） |

### 開始時の状況

> 「部活で疲れて家に帰ると、机に向かう気力が残らない」と相談を受けたのが始まりでした。
> 学校の提出物は半分ほどしか出せず、定期テストの成績も伸び悩む状況。本人も「やる気はあるけど続かない」と話していました。

### 3ヶ月後

> 毎日のチャットで進捗を確認する習慣ができ、週1回の面談で計画を調整するリズムが定着しました。
> 「やれた・やれなかった」を毎日報告するだけのシンプルな仕組みが、自分のペースを掴むきっかけになりました。

### 半年後

> 提出物の提出率が大きく改善。定期テストでも、特に英語と社会で点数が上昇しました。
> 大会期間中も計画を調整しながら勉強時間を確保できるようになり、「忙しくても続けられる自信がついた」と本人が振り返るようになりました。

### 1年後（指導終了時）

> **学校提出物の提出率：54% 向上（学期通算 92% 達成）**
> **定期テスト各科目平均点：+23点 向上**
>
> 内申点も着実に改善し、希望していた高校への進学準備が整いました。

### 本人コメント

> 「大会前の忙しい時期も、メンターと一緒に立てた計画のおかげで勉強を続けられました。
> 自分一人だと『今日はもういいや』ってなっちゃう日も、報告するから少しでもやろうって思えるのが大きかったです。」

### 担当コメント（中村龍人）

> ユウキくんは元々勉強への意欲はありましたが、忙しさの中で続けられる仕組みがなかったのが課題でした。
> 量を増やすのではなく、1日15分でも継続することを徹底し、それが結果として大きな変化につながりました。
> 「続けられる」という感覚が、本人の自信にもなったと感じています。

## 注記

- 物語性を保ちつつ、誇張表現は避ける
- 「やる気がある／ない」の二元論を避け、「続けられる仕組みがなかった」というフレーミング
- 担当コメントを入れることで、指導者の視点が見える

# セクション4：ケーススタディ詳細 - アヤカさん

ID: `#ayaka`

## 目的
アヤカさんの3ヶ月で何が起きたかを物語として伝える。

## レイアウト方針

- ユウキくんと同じ構造
- 時系列：開始時／1ヶ月後／2ヶ月後／3ヶ月後

## コピー

### 見出し
> アヤカさんのケース：高校3年生

### 基本情報

| 項目 | 内容 |
|---|---|
| 学年 | 高校3年生 |
| 状況 | 大学進学を目指して受験勉強中（指導開始時） |
| 指導開始 | 2025年4月 |
| 指導期間 | 3ヶ月 |
| 指導内容 | 週1回のオンライン面談（学習計画）／毎日チャットで進捗管理／オンライン個別指導 |
| 担当 | ヘッドコーチ・中村龍人（個別指導として） |

### 開始時の状況

> 受験を意識し始めたものの、「何をどう勉強すれば良いか分からない」状態でした。
> 問題集を開いても、答えを見ても何も分からずフリーズしてしまう。週あたりの勉強時間は3時間程度。学内順位も下位にとどまっていました。

### 1ヶ月後

> まずは1日30分から始める計画を立て、毎日チャットで進捗を報告する習慣を作りました。
> 「分からない問題は写真で送る」ルールを徹底し、つまずきをその場で解消する流れが回り始めました。

### 2ヶ月後

> 勉強時間が週10時間程度まで増加。学内テストでも明確な変化が出始めました。
> 本人が「最近、勉強が楽しいかもしれない」と言うようになったのが、この時期です。

### 3ヶ月後（指導終了時）

> **週あたり勉強時間：470% 向上（週 3h → 14.5h）**
> **学内テスト順位：約 500位 向上**
>
> 受験勉強の本格化に向けて、自走できる状態が整いました。

### 本人コメント

> 「最初は問題集の答えを見ても何もわからずフリーズする状態でしたが、
> だんだん勉強する習慣ができて楽しくなりました。
> メンターが毎日応援してくれるので、続けることが苦じゃなくなりました。」

### 担当コメント（中村龍人）

> アヤカさんは「勉強の仕方が分からない」状態からのスタートでした。
> 最初の1ヶ月は学習量よりも「毎日机に向かう習慣」を作ることに集中しました。
> 習慣ができると、自然と量も増え、結果がついてきます。短期間ですが、本人の意識が大きく変わったケースです。

## 注記

- ユウキくんと違って短期間（3ヶ月）。中高生で習慣化と意識変化に必要な期間の参考例
- 「勉強が楽しいかもしれない」というセリフは引用として印象的
