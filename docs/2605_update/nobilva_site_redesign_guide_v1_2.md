## 4. 共通の設計原則（全ページ共通）

### 4-1. ビジュアル
- **Adobe Firefly Web 生成画像で統一**（Nanobanana・Adobe Stock は廃止）
- トーン：明るいライフスタイル写真風／自然光（昼〜午後）／顔なし／顔出さない（後ろ姿・手元・部分・シルエット）
- ブランドカラー：黄色 `#f6ce4a`（主色）／オレンジ `#ee741a`（補助色）
- 青 `#2c3e7a` は使用しない（旧設計の名残・現方針では誤り）

### 4-2. レイアウト・実装
- **モバイルファースト**で組む（主読者は夜にスマホで閲覧する保護者）
- iPhone SE（375px）幅で破綻しないこと
- 横長要素（比較表など）はモバイルで横スクロール可
- プランカード・主要カードは縦並び（横スワイプはなし）
- アスペクト比はモバイル縦持ちでも切れないものを優先

### 4-3. 主語ルール（重要）
**生徒（子ども）側中心の地の文。「お子さん」「我が子」「本人」は明示しない**。
日本語の主語省略を活かす。

例外：以下のセクションのみ親主語・「お子さん」許容
- LP セクション2 共感ブロック（葛藤がテーマ）
- LP セクション9 コーチメッセージ（メッセージ性）
- 各FAQセクション（保護者向け説明文として中立）

### 4-4. 用語ルール（厳守）

**OK**
- サービスコピー：「野球をがんばる中高生のための学習管理サービス」
- 三本柱：「日割り学習計画／週1回オンライン面談／毎日の進捗確認」
- リーグ名：リトルシニア・ボーイズリーグ・ポニーリーグ・ヤングリーグ・中学硬式（部活）・中学軟式（部活）・高校野球部

**NG**
- 「少年野球」「球児」「スポ少」「少年」
- AI分析・親ダッシュボード・月次レポート・月次コーチチェックインへの言及

### 4-5. CTA戦略
- **プライマリCTA：必ず `/diagnosis`** を向ける
- **LINE導線は副次扱い**（フローティング・フッターのみ・プライマリにしない）
- 各サブページから他サブページへの **セカンダリCTA** は文脈に応じて配置

### 4-6. SEO/AEO
- 各ページに H1 は1つだけ
- 各サブページの章末FAQに **構造化データ（FAQPage）** を実装
- メインページに **Service** 構造化データ
- 進路ガイドに **Article** 構造化データ
- 料金ページに **Product / Offer** 構造化データ
- 指導者紹介に **Person** 構造化データ（オプション）

## 5. 主要意思決定事項のサマリー

### 5-1. 価格・割引
- エッセンシャル：**18,000円/月**（全科目対応）
- ベーシック：**26,000円/月**（全科目対応・おすすめ）
- 個別指導オプション：**仮料金**（週1コマ 20,000円〜／週2コマ 35,000円〜／週3コマ 50,000円〜）— **正式確定はTODO**
- 兄弟2人目：**月額10%割引**
- チーム経由：**通常価格から10%割引（仮設定）** — 正式確定はTODO
- **紹介手数料は提供しない**（チーム経由の紹介に対する還元は、保護者の月額割引として直接ご家庭に反映する形に統一）

### 5-2. 支払い・契約
- 支払い方法：**銀行振込のみ**（クレジットカード非対応）
- 振込手数料：**お客様負担**
- 初月：**日割り計算**
- 入会金：**0円**
- 解約：**前月25日まで連絡で翌月解約**
- 違約金：**なし**
- 一時休会：**1ヶ月単位で可能**

### 5-3. 30日全額返金保証
- 入会から30日以内の申し出で全額返金
- 申請方法：`nobilva@nectere.jp` まで件名「30日返金保証 申請」
- 返金タイミング：受領から7営業日以内
- 返金対象外：教材費・30日超過分
- 規約整備は Week 4 タスク（ピボット計画書 §11）／訴求は初期から実施

### 5-4. オファー
- **月20名限定 無料学習診断**（30分・Zoom・無料）
- LINE登録不要
- 当日申込み強要なし

### 5-5. メンター体制
- **ヘッドコーチ・中村龍人**（東京都立戸山高校卒・東京大学計数工学科 2024年入学・中学時代FLL・高校時代ブラスバンド3年9月まで）
- **代表メンター・養田貴大**（プロフィール情報は要記入・現時点で [要記入] プレースホルダー）
- 段階2：東京大学野球部OB顧問配置（時期未定）
- 段階3：本格メンター採用（時期未定）
- 担当メンターは原則継続（無料変更は約束しない）

### 5-6. 連絡先
- メール：`nobilva@nectere.jp`
- 電話：`03-6820-9037`
- 担当：中村龍人・養田貴大

## 8. Claude Code への申し送り事項

### 8-1. ファイルの読み方
1. 各設計書の **「0. 設計の前提」** に主読者・温度・やらないことが定義されている
2. 各セクションには **目的・レイアウト方針・コピー・注記** が記述されている
3. **コピーは そのまま実装してよい** 形で書かれている（Claude Code 側での創作は不要）

### 8-2. 共通実装事項
- **ヘッダー**：メインLPと各サブページで共通ナビ（特徴／料金／進路ガイド／チーム／お問い合わせ）
  - 例外：`/diagnosis` はナビ非表示（離脱防止）・LPに戻る1リンクのみ
- **フッター**：全ページ共通（サイトマップ／法的情報／運営者／SNS）
- **30日返金保証バッジ**：LPヒーロー・LP料金プラン・LP最終CTA・各サブページCTA直前 に配置
- **月20名限定バッジ**：LPヒーロー・LP最終CTA・`/diagnosis` ヒーロー に配置

### 8-3. 注意事項
- **コピーの改変は最小限に**（既に各ページで主語ルール・用語ルールを徹底してチェック済み）
- **数字は設計書記載のまま実装**（勝手な変更は不可、ユーザー確認なしでは変えない）
- **競合への言及**：LP セクション8 比較表・`/career-path` セクション8 学習支援サービス選び — 既に中立トーンで書いてあるので、改変不要
- **「お子さん」明示**は許容セクション以外では使用しない

### 8-4. 構造化データの実装
- 各章末FAQ・全件FAQページ：**FAQPage**
- メインLP：**Service**
- `/career-path`：**Article**
- `/pricing`：**Product** + **Offer**（プランごと）
- `/coach`：**Person**（オプション）
- `/results`：**Review**（オプション・仮名のため Person は不要）
