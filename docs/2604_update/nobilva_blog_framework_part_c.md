## C0. テンプレートの使い方

### 作業順

1. テンプレを記事ごとに複製（`記事スラッグ.md` でファイル化推奨）
2. C1 〜 C7 の項目を上から順に埋める
3. 完成した記入済みテンプレを Claude Code に渡して本文生成
4. 生成された本文を編集・調整
5. C8 で画像プロンプト→ Nanobanana で画像生成
6. C9 のチェックリストで公開前確認
7. 公開後、Part B のステータスを「公開済み」に更新

### 所要時間の目安

- C1 〜 C7 の記入：30〜60分
- Claude API での本文生成：5〜10分
- 編集：60〜120分
- 画像生成・選定：30〜60分
- 合計：1記事あたり3〜5時間

# テンプレ本体

## C1. 記事の位置づけ

- **記事スラッグ**：→ 英訳でハイフン区切り（例：`sports-recommendation-eiken-requirements`）
- **記事タイトル（仮）**：→ A10 のルールで作成、後で確定
- **クラスタ軸**：→ 時間／進路／自己認識／関係性のいずれか
- **記事タイプ**：→ ハブ／子記事
- **主読者**：→ 保護者向け／生徒向け／両方向け
- **ボリューム**：→ 軽量（2,000〜3,000字）／標準（3,500〜5,000字）／重量（6,000〜8,000字）
- **配置パターン**：→ パターン1（シンプル3枚）／パターン2（リッチ5〜6枚）／パターン3（最小1〜2枚）

# パイロット記入例：進路軸ハブ記事

> 進路軸ハブ記事を実例として記入する。実際の運用イメージを掴むためのサンプル。

## C1. 記事の位置づけ

- **記事スラッグ**：`baseball-student-career-path-guide`
- **記事タイトル（仮）**：野球をがんばる中高生の進路の選び方 — スポーツ推薦・指定校・一般入試の現実 \| Nobilva
- **クラスタ軸**：進路軸
- **記事タイプ**：ハブ記事
- **主読者**：保護者向け
- **ボリューム**：重量（6,000〜8,000字）
- **配置パターン**：パターン2（リッチ5〜6枚）
- **公開予定日**：Month 1 末
- **新規 or リライト**：新規

## C2. キーワード設計

- **メインキーワード**：野球 進路（または野球部 進路）
- **関連キーワード**：スポーツ推薦／指定校推薦／野球 一般受験／文武両道 高校／野球部 進学
- **共起語**（ラッコキーワード裏取り後に確定）：内申点／評定／英検／推薦基準／併願
- **想定検索者像**：中学硬式または中学野球部に所属する子をもつ中3〜高1の保護者。スポーツ推薦も視野にあるが、一般受験の選択肢も残したい。具体的にどう動けばいいか整理して理解したい。
- **検索意図**：情報収集（俯瞰したい）
- **競合上位記事の傾向**：大手塾（栄光ゼミナール・スタディサプリなど）の汎用記事が並ぶ。野球特化視点で書かれたものは少ない。差別化ポイントは「野球家庭固有のスケジュール感（試合期・引退期）」と「内申点との両立の具体策」。

## C3. リード文（200〜300字）

```
中学・高校で野球を続けるお子さんの進路は、スポーツ推薦・指定校推薦・公募推薦・総合型選抜・一般入試と、選択肢が複数ある一方で、それぞれに準備のタイミングと必要な条件が異なります。野球の練習スケジュールと両立できる現実的な動き方を、保護者の方が把握しておくことで、選択肢を残したまま準備を進められます。本記事では、野球をがんばる中高生の進路選択を、入試の種類・必要な準備・両立のポイントの3つの軸で整理しました。
```

## C4. 章立て（マズロー5段階）

### 1章（生理的欲求 — 現実の物理的問題）
- **章タイトル**：野球をがんばる中高生の進路選択で、最初に直面する現実
- **要旨**：練習時間・遠征・大会期で勉強時間が圧倒的に削られる現実。中3夏まで部活、引退後3〜6ヶ月で受験という時間制約。保護者が陥りやすい「とりあえずスポーツ推薦」一択思考の危険性。
- **盛り込む情報**：中学硬式の年間練習スケジュール感、引退時期の地域差、受験までの実勉強時間の概算

### 2章（安全欲求 — 不安の言語化）
- **章タイトル**：「進路の選択肢が狭まるのでは」という不安の正体
- **要旨**：保護者が抱える4つの典型的な不安（成績低下／推薦の不確実性／一般受験への対応力／お子さん本人の意向）を分解し、それぞれ何が原因で何が解決可能かを整理。
- **盛り込む情報**：スポーツ推薦の合格率の現実（学校により幅があること）、指定校推薦の枠数の限定性、内申点の持つ意味

### 3章（所属欲求 — 同じ立場の家庭の存在）
- **章タイトル**：野球家庭が選びやすい進路パターン3つ
- **要旨**：実際に野球家庭が辿りがちな3つの進路パターン（A: スポ推一本／B: 推薦＋一般併願／C: 引退後一般受験切替）を紹介。それぞれの家庭がどんな判断基準で選んだかを描写。
- **盛り込む情報**：3パターンの具体的な動き方、各パターンの向き不向き、ケーススタディ（仮名）
- **ケーススタディの有無**：あり（3パターン × 1家庭ずつ仮名で）

### 4章（承認欲求 — 解決の方向性と具体策）
- **章タイトル**：選択肢を残しながら進路準備を進めるための4つのステップ
- **要旨**：①情報収集（学校別の推薦基準・内申要件）②学習計画の構築（部活と並行できる現実的な時間配分）③英検・評定の戦略的取得 ④進路決定のタイミング設計、の4ステップで具体的アクションを提示。
- **盛り込む情報**：英検取得の標準スケジュール、評定平均の積み上げ方、進路決定タイミングの目安
- **内部リンク**：
  - スポーツ推薦 対策（リライト後の野球特化版）
  - スポーツ推薦 英検
  - スポーツ推薦対策 いつから
  - 部活引退後 勉強
  - 野球 内申点 高校（裏取り後採用なら）

### 5章（自己実現 — 最終的なゴール像）
- **章タイトル**：「お子さんが自分で選べる」状態をつくる進路設計
- **要旨**：保護者が代わりに決めるのではなく、お子さん本人が複数の選択肢を持って自分で選べる状態が理想。そのために必要な準備期間と家族の関わり方を提示。
- **Nobilva の自然な位置づけ**：「家庭だけで全部背負わない選択肢として、学習管理を外部に任せる方法もある」という流れで自然に言及

### FAQ 章
- **質問1**：スポーツ推薦と一般入試はどちらの準備を優先すべきか
- **回答1**：併願準備を基本とし、中3夏までに英検と評定平均を確保したうえで、引退後に重点をシフトするのが現実的（記事本文と整合させた回答）
- **質問2**：野球部活動中に英検は何級まで取っておくべきか
- **回答2**：志望校の基準による。一般的には準2級〜2級が目安だが、私立強豪校の中には準1級を求める学校もある（具体例を記載）
- **質問3**：内申点はいつから意識すべきか
- **回答3**：中1の1学期から。中3だけ頑張っても挽回が難しい仕組みであることを説明
- **質問4**：野球を続けながら塾に通えない場合、どんな選択肢があるか
- **回答4**：オンライン家庭教師・通信教材・学習管理サービスなどの選択肢を中立的に提示し、Nobilva は学習管理サービスのカテゴリに位置づけ
