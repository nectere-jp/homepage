# Nobilva 戦略・タスクリスト（野球特化リブランド）

最終更新: 2026年4月26日
作成: 中村龍人 × 養田貴大

---

## 1. 方針

### コアテーゼ

**野球をがんばる中高生のための学習管理サービス**として再ポジショニングする。
直接競合 STUDY ATHLETE がサッカー特化で取りに来ているのと同じ手法を、競合不在の野球領域で取りに行く。

### ターゲット

- 小学6年生〜高校3年生
- リトル・シニア・ボーイズ・ポニー所属の小6〜中3
- 中学硬式・軟式野球部の中高生
- 強豪校野球部の高校生
- スポーツ推薦・内申点・部活両立に課題感を持つ家庭

### 戦い方

- 月18,000円〜の価格帯はそのまま維持
- 「30日全額返金保証」でリスクゼロを訴求
- 当面は B2C を太くする（リトルシニア系チーム＋強豪校父母会経由）
- 月10万円規模のメディア出稿は不採用、人的リソースと地道な営業で積む

## 4. 無料学習診断オファー

### 仕組み

- 専用ページ（アンケート＋日程調整一体型）
- LINE 追加なし、フォーム完結
- チーム LINE グループに直接リンク投下してもらう
- 月20名限定（希少性訴求＆工数管理）

### アンケート設問

- 学年・性別
- 部活・所属チーム名
- 週の練習時間・休日
- 志望校 / 進路イメージ（推薦狙いか・一般か・未定か）
- 現在の評定（5段階 × 9教科）
- 苦手科目・得意科目
- 直近の定期テスト点数
- 1日の勉強時間
- 親の連絡先（任意）
- 希望面談日程（カレンダー連携）

### 1人あたり工数（目標）

- 事前読み込み：15分
- 面談：30分（厳格に）
- レポート作成：15分（テンプレ80%流用）
- 合計：50〜70分

月20名で月20時間 ≒ 週5時間。死なないライン。

### 実装スタック

- Next.js + Supabase（既存）
- カレンダー連携：Google Calendar API or Cal.com 埋め込み
- 自前実装を推奨（既存スタックに統合）

## 8. 営業チャネル

### B2C（最重要）

- LP 経由問い合わせ
- 商店チラシ・ビラ
- 既存ブログ SEO（継続、撤退はしない）

### B2B(2C)

**(a) リトルシニア・ボーイズ・シニア・ポニー（営業継続）**

- 既存営業先のチーム LINE グループにアンケートリンク投下
- 1〜2家庭契約 → 紹介手数料制で他家庭に拡散
- 紹介手数料：1家庭契約成立で5,000〜10,000円

**(b) 強豪校野球部の父母会・後援会（新規）**

- 「東大生による学習法講座（無料）」をオファー
- 1校で50〜100人の保護者にリーチ可能
- 窓口：父母会会長・広報担当の保護者
- ターゲット校リスト30校を作成（東京・千葉・神奈川）
- **アプローチは外部営業中心（共通人脈経由は当面なし、個人ネットワーク非依存）**
- 連絡先源：学校公式サイトの後援会ページ、地方高野連、強豪校紹介メディア、Twitter/X の野球部公式アカウント

### やらない（当面）

- 高校校長・部長・顧問への直接アタック
  - 実績10チーム後まで保留
  - 属人的・前例主義・決裁通らない・メリット薄い

## 9. アナリティクスツール（Claude Code で内製）

### 要件

- テレアポ進捗・LINE 追加数・ページ表示数を一元管理
- 既存テレアポツールと連携可能
- スクショで簡単に反映できる仕組み

### 入力データ

- テレアポ進捗（**養田が Cursor で自作したツールと連携、DB スキーマ共有可**）
- LP セッション数・流入チャネル（GA4 API or スクショ）
- LINE 友だち追加数（LINE Messaging API or スクショ）
- 無料診断予約数（自社 DB）
- 契約件数（自社 DB）

### 機能

- ファネル可視化（流入→アンケート→面談→契約）
- チャネル別 ROI
- ボトルネック自動ハイライト
- 週次サマリ通知

### 実装優先順

1. 手入力フォーム＋ Supabase DB（1日）
2. ファネル可視化ダッシュボード（2〜3日）
3. 既存テレアポツール連携（1日、API 仕様確認後）
4. スクショ OCR 対応（必要性が出たら）
5. GA4 / LINE API 自動取得（後回し）

### スタック

- Next.js（既存）
- Supabase（既存、RLS で社内ユーザー制限）
- shadcn/ui（既存）
- Recharts（チャート）

## 10. 事業形態

### 現状

- **屋号付き個人事業主**として登記済み
- 屋号：Nectere
- 代表：養田貴大
- 中村は登記なし（業務委託・共同運営の立場）

### 当面の方針

- 個人事業主のまま運営継続
- 法人住民税（赤字でも年7万円）等の固定費を発生させない
- 青色申告（65万円控除）で運用
- 特定商取引法表記は「Nectere（屋号）／代表 養田貴大」のまま

### 法人化の検討タイミング

下記のいずれかが起きたら合同会社化を再検討：

1. 月商30万円超が3ヶ月継続
2. 大手企業・学校法人との直接契約が必要になる
3. チーム導入で年契約100万円超が見える
4. 投資検討（株式会社化）

### 中村の役員報酬・扶養

- 中村は何も登記していないため法人化されるまで関係なし
- 業務委託報酬を受け取る場合、丸亀＋成城塾＋Nobilva 業務委託 = 年130万円未満を維持
