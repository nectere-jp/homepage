# Nobilva ブログ戦略：キーワード候補リスト（裏取り用）

最終更新：2026年4月28日
位置づけ：ピボット後（野球特化）のブログクラスタ設計のための**前段ドキュメント**

---

## 0. このドキュメントの使い方

このドキュメントは、フレームワーク本体（Part A/B/C）を作る前の**キーワード裏取り用の作業台帳**。

### 使い方の流れ

1. このドキュメントを Claude Code と一緒に読む
2. 各候補キーワードをラッコキーワードに投入し、関連語・共起語・月間検索ボリュームを確認
3. 採用判断（保留も可）
4. 採用キーワードを CMS の「キーワード管理」に追加・既存キーワードのうち削除対象を整理
5. 採用キーワードが固まったら、**フレームワーク本体（Part A：再利用基盤、Part B：クラスタ管理、Part C：記事個別）と Claude API プロンプトテンプレートの作成フェーズに進む**

```
時間軸（大）
 └ ハブ：「野球と勉強の両立に悩む保護者の方へ」
   └ 子記事：練習で時間がない／遠征／オフシーズン／オンライン指導 など

   └ 子記事：自信喪失／苦手克服／強豪校で挫折／計画が続かない など

   └ 子記事：声かけ／夫婦の温度差／父母会／兄弟比較 など

 └ メガスタ破産（既存記事温存）
```

## 5. 主読者別のトーン方針（記事ごと使い分けの指針）

### 保護者向け記事の特徴

- 一人称「お子さん」「ご家庭」表記
- 共感→情報提供→提案 の構成
- 受験・進路・評定・推薦などの**情報精度**が重要
- CTA は「無料学習診断」を保護者目線で訴求
- トーン：温かく丁寧、Instagram の延長線

### 生徒向け記事の特徴

- 一人称「自分」「あなた」表記
- 共感→具体的アクション の構成
- ノウハウ・体験談・**実行可能性**が重要
- CTA は「親に相談してみる」「無料学習診断を一緒に見てもらう」
- トーン：親しみやすく、上から目線にならない

### 両方向け記事

- リード文で「保護者の方も、本人の方も」と明示
- 章ごとにどちらに語りかけているかを明確化
- 結論はどちらの読者にもアクション可能なものに

## 6. ラッコキーワード裏取りの作業手順

各候補キーワードについて以下を確認：

1. **関連キーワード・共起語の確認**
   - 候補キーワード周辺で実際に検索されている語を把握
   - 想定外の意図が混じっていないかチェック（例：「野球 親 つらい」が指導者批判系に偏っていないか等）

2. **検索ボリュームの目安**
   - ロングテール（〜100/月）でも CV 意図が深ければ採用
   - ボリュームゼロは保留。ただし関連語に展開する余地があれば残す

3. **競合状況の確認**
   - 上位記事が大手塾・教育メディアばかりなら正面突破は不利
   - ロングテールや具体性で差別化できる切り口を探す

4. **採用判断の基準**
   - **採用**：CV 期待値が高い／既存キーワードと相互補強できる／クラスタ構造の穴を埋める
   - **保留**：意図が混在／競合が強すぎる／類似キーワードに統合可能
   - **不採用**：野球特化と矛盾／CV 動線が見えない

5. **採用キーワードを CMS に追加**
   - タグ付け（時間軸／進路軸／自己認識軸／関係性軸／緊急受け皿）
   - 主読者ラベル（保護者／生徒／両方）
   - 優先度（高／中／低）

## 8. KPI（毎月見る指標）

ブログ単体の成果指標：

1. **無料学習診断申し込み数**（最重要KPI）
2. ブログ → Nobilva LP 遷移数
3. ブログ → 無料学習診断ページ遷移数
4. クラスタ別 PV
5. クラスタ別 CTR（GSC）
6. CV につながった記事の特定（GA4）
7. 公開記事数（運用指標）

## 9. やらないことリスト（明示）

- 非野球分野（サッカー・バスケ・吹奏楽など）の新規記事
- メガスタ破産記事のリライト・拡張（独立カテゴリのまま温存）
- ビッグワード正面突破（「家庭教師」「中学生 勉強」など）
- ボリューム狙いの量産記事
- Instagram と完全に重複する感情共感系記事（住み分け重視）
