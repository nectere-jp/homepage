# Nobilvaブログ見出しスタイル - 実装完了

## 概要

Nobilvaブランドに合わせた、モダンでボールドな見出しスタイルを実装しました。
各見出しレベルに応じて、グラデーション、背景色、ホバーアニメーションを適用しています。

## 実装された見出しスタイル

### H1 - メインタイトル

- **背景**: 黄色からオレンジへのグラデーション (#f6ce4a → #ea5614)
- **テキスト**: 黒、font-black (900)
- **サイズ**: 3xl (モバイル) / 4xl (デスクトップ)
- **パディング**: 1.5rem 2rem
- **シャドウ**: 0 6px 12px rgba(0, 0, 0, 0.15)
- **ホバー効果**:
  - Y軸に-3px移動
  - シャドウ拡大: 0 8px 16px rgba(0, 0, 0, 0.2)

### H2 - 主要セクション

- **背景**: オレンジのグラデーション (#ea5614 → #bb4510)
- **テキスト**: 白、font-black (900)
- **サイズ**: 2xl (モバイル) / 3xl (デスクトップ)
- **左ボーダー**: 6px solid #f6ce4a (黄色)
- **パディング**: 1rem 1.5rem
- **シャドウ**: 0 4px 8px rgba(0, 0, 0, 0.12)
- **ホバー効果**:
  - Y軸に-2px移動
  - シャドウ拡大: 0 6px 12px rgba(0, 0, 0, 0.18)

### H3 - サブセクション

- **背景**: 黄色 (#f6ce4a)

## デザインコンセプト

1. **ブランド統一**: Nobilvaのメインカラー（黄色）とアクセントカラー（オレンジ）を使用
2. **視覚的ヒエラルキー**: 見出しレベルが一目でわかる明確な差異
3. **モダン＆ボールド**: 強いコントラスト、グラデーション、シャドウ効果
4. **インタラクティブ**: ホバー時のアニメーションで動的な印象
5. **レスポンシブ**: モバイルとデスクトップで適切なサイズ調整

## 技術仕様

- **トランジション**: all 0.3s ease
- **角ばったデザイン**: border-radiusなし（Nobilvaブランドに合わせる）
- **アクセシビリティ**: 十分なコントラスト比を確保
- **パフォーマンス**: CSS transformを使用してGPU加速

## 適用範囲

- **対象クラス**: `.prose-nobilva`
- **適用ページ**: `relatedBusiness: ["nobilva"]` が設定されたブログ記事
- **ファイル**: `/app/globals.css`

## テスト方法

1. 開発サーバー起動: `npm run dev`
2. サンプル記事にアクセス: `http://localhost:3000/ja/blog/sample-blog-post`
3. 各見出しレベル（H1～H6）の表示を確認
4. ホバー時のアニメーション動作を確認
5. モバイル・デスクトップの両方で確認

## サンプル記事

`/content/blog/sample-blog-post.md` に以下の見出しが含まれています：

- H1: メインタイトル
- H2: 主要セクション（複数）
- H3: サブセクション（複数）
- H4以下: 詳細項目

## 互換性

- **デフォルトテーマ**: 影響なし（`.prose-default`）
- **TeachItテーマ**: 影響なし（`.prose-teachit`）
- **既存記事**: すべて互換性あり（Markdownコンテンツ変更不要）
